# PROCESOS
  * Creación de proyecto con vite via npm
    ```
    npm create vite@latest
    ```
    ```
    cd my-project
    ```
    ```
    npm install
    ```
    ```
    npm run dev
    ```

  * Instalación de Cypress
    ```
    npm install cypress --save-dev
    ```
    * Agregar la siguiente linea en nuestro package.json
      ```
      "cypress": "cypress open"
      ```
